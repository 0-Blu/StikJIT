Project Overview

StikDebug is a cutting-edge debugging and just-in-time tool built for iOS devices. Combining high-performance technology with an intuitive, SwiftUI-inspired interface, StikDebug streamlines installation, usage, and troubleshooting.

Features
  • Seamless installation via AltStore PAL
  • Modern, dark mode user interface
  • Robust pairing and debugging features
  • Detailed log viewing with Open Console
  • Integrated support for shortcuts and automated actions
Instructions
How to Install StikDebug (with AltStore PAL)
  1. Get AltStore PAL installed: go to altstore.io and follow the instructions.
  2. Add the StikDebug source to AltStore PAL: https://stikdebug.xyz/apps.json.
  3. Install it via AltStore PAL – and now you're done!
Pairing File Instructions
  1. Make a pairing file and follow the guide at SideStore Pairing File Guide.
  2. After creating the pairing file, zip it up and share it with your device. We recommend emailing the file to yourself or uploading it to a cloud service like Airdrop or Proton Drive.
How to Use StikDebug
  1. Open the app, read the popup, and click "Continue" to enable the loopback VPN.
  2. Connect to an app process by clicking the button named "Connect" and selecting an app from the list.
How to View the Logs

Click the Open Console button to view the logs.

Common Issues
  1. Pairing File Issue -9
    Info: This may occur if your pairing file was tampered with or if you created a brand new one.
    Fix: Make a new pairing file with JitterBugPair.
FAQ
  • Do I need to be connected to WiFi?
    WiFi or Airplane mode is required.
  • Is this open source?
    Yes – check out the source code on GitHub.
  • What iOS versions does this support?
    iOS 17.4 – Latest (latest version).
  • Will anything below 17.4 work?
    No, update your iOS version.
